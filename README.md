# DVD Bouncing

A recreation of the classic **DVD Screensaver** logo bouncing around the screen using _AABB Collision Detection_ method, built in **Lua** and **LÖVE2D**.

# Preview

![DVD Bouncing](./dvd.gif)

# Running

You have to have [LÖVE2D](https://love2d.org/) installed, then

clone the project

```bash
git clone https://github.com/HsHs-dev/dvd-bouncing-logo.git
```

navigate to the project directory

```bash
cd dvd-bouncing-logo
```

and run it using the following command

```bash
love .
```

> [!NOTE]
> Press `space` or `escape` to quit.
> If you would like to change the logo _opacity_ or moving _speed_, change `SPEED` or `alpha` in `main.lua`
